## Tarefas e Provas

### Abertas
``` dataview
TASK
WHERE icontains(text, this.file.name)
AND (icontains(text,"#task") OR icontains(text,"#question"))
AND !completed
GROUP BY file.name as filename
SORT filename Desc
```

### Fechadas
```dataview
TASK
WHERE icontains(text, this.file.name)
AND (icontains(text,"#task") OR icontains(text,"#question"))
AND completed
GROUP BY file.name as filename
SORT filename DESC
```

### Provas
```dataview
TASK
WHERE icontains(text, this.file.name)
AND icontains(text,"#log/sprint")
GROUP BY file.name as filename
SORT filename DESC
```
